# 🚨 IoT-Based Smart Accident Detection and Emergency Alert System

## 📌 Overview

This project is an IoT-based system designed to automatically detect accidents and send real-time alerts with location details.

## 🚀 Features

* Automatic accident detection using MPU6050 sensor
* Real-time alert system
* GPS-based location sharing
* Telegram bot integration
* Works without user interaction

## 🧠 Working

1. Sensor continuously monitors motion
2. Sudden impact is detected
3. System identifies accident
4. GPS fetches location
5. Alert is sent via Telegram

## 🧩 Components Used

* NodeMCU (ESP8266)
* MPU6050 Sensor
* GPS Module
* LED Indicator
